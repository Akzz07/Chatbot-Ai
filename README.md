# Temp AI Chatbot

Modern neon-themed AI chatbot built with Node.js & Groq API.

## Features

- Neon futuristic UI
- Groq LLM integration
- Fast response
- Clean architecture (routes / services)
- Secure environment config

## Tech Stack

- Node.js
- Express
- EJS
- Groq API
- Vanilla CSS (Neon Style)

## Environment Setup

Create `.env` file:

```env
GROQ_API_KEY=your_api_key_here
```
